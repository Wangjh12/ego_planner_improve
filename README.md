### 改进说明

在原版 EGO‑Planner 的无 ESDF 梯度优化框架基础上，本分支针对在线轨迹规划在飞行时间效率与动态环境响应方面的不足，引入以下三阶段改进：

#### 1. 时间优化
- 在目标函数中加入时间惩罚项，直接缩减任务总耗时  

#### 2. FOV 约束
- 构建几何可见性模型，将传感器视场（FOV）限制嵌入轨迹生成  

#### 3. 偏航角规划
- 基于 B 样条参数化的偏航角规划方法
